## Known gotchas

### `better-sqlite3` ABI bouncing

Node 24 uses `NODE_MODULE_VERSION 137`. Electron 41 uses `145`. The `postinstall` hook rebuilds for Electron ABI, breaking Vitest. Run `npm rebuild better-sqlite3` to switch back to Node ABI.

### Headless verification

No GUI verification possible in this harness. A human should boot `npm run dev`, configure a real API key, and smoke-test the recording → note → edit → search flow.

### Sidecar URL placeholders

`scripts/fetch-sidecars.mjs` and `src/main/whisper/registry.ts` contain `REPLACE_WITH_…` placeholders. Fill in verified URLs and SHA-256 hashes before packaging or running the full pipeline.
